html basics 

<!DOCTYPE html> — tells the browser this is an HTML5 document  
<html> — root of the page  
<head> — metadata, links, title, etc.  
<title> — title shown in the browser tab  
<body> — main content of the page

text elements

<h1> to <h6> — headings from most to least important  
<p> — paragraph  
<br> — line break  
<strong> — bold text (semantic)  
<em> — italic text (semantic)  
<b> — bold (non-semantic)  
<i> — italic (non-semantic)

structure elements

<div> — generic block container  
<span> — generic inline container  
<section> — standalone section  
<article> — self-contained content  
<main> — main content of the page  
<header> — top of a section/page  
<footer> — bottom of a section/page  
<nav> — navigation links

links & images

<a href="url"> — hyperlink  
target="_blank" — open in new tab  
<img src="path" alt="desc"> — image tag  
width="" height="" — control image size  
alt — alternative text (for screen readers and if image fails)
